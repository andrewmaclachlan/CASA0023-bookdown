Welcome

Hello and welcome to the Term 2 module Remotely Sensing Cities and Environments.

Similarly, to my Term 1 MSc module, CASA0005, this website holds all the practical instructions for the module. CASA0005 Geographic Information Systems and Science is a pre-requisite of the module so concepts taught there will mainly be assumed here.
